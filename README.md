# Asset Organizer

드라이브 적재 전 소재 파일·폴더를 정리하는 GUI 툴.
네이밍 변경, 사이즈 매칭, 폴더 그룹핑을 코드 수정 없이 규칙 입력만으로 처리.

---

## 📋 기능

| 모드 | 설명 |
|------|------|
| **전/후 예시 입력** | 예시 1쌍 입력 → 변환 규칙 자동 추론 후 일괄 변경 |
| **찾기 / 바꾸기** | 찾을 문자열 + 바꿀 문자열 직접 입력 |
| **사이즈 매칭** | 기준 파일 이름을 따라가되 사이즈(숫자x숫자)만 교체. 폴더명 변경 옵션 포함 |
| **폴더 그룹핑** | 사이즈만 다른 같은 소재 파일들을 하나의 폴더로 묶기 |

---

## 🚀 설치 및 실행

### 요구사항

- Python 3.8 이상
- 추가 설치 불필요 (표준 라이브러리만 사용)

### 실행 방법

```bash
python asset_organizer.py
```

또는 파일 탐색기에서 `asset_organizer.py` 더블클릭.

> **Windows:** Python 설치 시 "Add to PATH" 옵션을 체크해야 더블클릭 실행이 가능.

## 📖 사용법

### 공통 순서

```
① 작업 방식 선택
② 규칙 입력
③ 대상 폴더 선택
④ 변경 범위 선택
⑤ 🔍 미리보기 → 결과 확인
⑥ ▶ 변경 실행
⑦ (필요 시) ↩ 실행 취소
```

---

### 1. 전/후 예시 입력

예시 1쌍을 입력하면 변환 규칙을 자동으로 추론.
**네이밍 규칙이 바뀌어도 코드 수정 없이 예시만 새로 입력**

```
변경 전: 260530_Asachuru_77-sakura_VID_01_Megawari_1080x1920_JP
변경 후: 260530_Asachuru_77-sakura-INF_VID_01_Megawari_1080x1920_JP
```

→ `_VID 앞에 -INF 삽입` 규칙으로 추론. 같은 패턴의 다른 파일 전체에 적용.

> 이미 변환된 파일(`-INF` 포함)은 자동으로 건너뜀.

---

### 2. 찾기 / 바꾸기

```
찾을 문자열: Megawari
바꿀 문자열: Supersale
```

- "이름당 첫 번째 일치만 변경" 옵션으로 중복 치환 제어 가능

---

### 3. 사이즈 매칭

정방향(1080x1080) 파일의 네이밍은 완료됐는데, 나중에 받은 다른 사이즈 파일이 제각각 이름일 때 사용.

#### 자동 (권장)

폴더 안 `1080x1080` 파일을 기준으로 나머지 사이즈를 자동 변환.

```
기준:    260517_..._Makeup-grip-INF_VID_04_evergreen_1080x1080_JP.mp4
새 파일: raw_export_세로형_1080x1920.mp4
  → 260517_..._Makeup-grip-INF_VID_04_evergreen_1080x1920_JP.mp4

새 파일: 배너용 1200x628.mp4
  → 260517_..._Makeup-grip-INF_VID_04_evergreen_1200x628_JP.mp4
```

- **폴더명도 함께 변경** 옵션 체크 시: 폴더명이 기준 파일명과 다르면 폴더명도 함께 변경
- 하위 폴더 포함 체크 시: 여러 소재 폴더를 한 번에 처리

#### 수동 (1:1)

정방향이 없거나 예외 상황일 때. 새 파일 1개 + 기준 파일 1개를 직접 선택.

---

### 4. 폴더 그룹핑

사이즈만 다른 같은 소재 파일들을 하나의 폴더로 묶음.
**새 폴더명 = 그룹의 1080x1080 파일명 (확장자 제외)**

```
[변경 전 — 한 폴더에 섞여있음]
260517_..._1080x1080_JP.mp4
260517_..._1080x1920_JP.mp4
260517_..._1200x628_JP.mp4
260520_..._1080x1080_JP.jpg
260520_..._1080x1920_JP.jpg

[변경 후 — 소재별 폴더로 묶임]
260517_..._1080x1080_JP/
  ├── 260517_..._1080x1080_JP.mp4
  ├── 260517_..._1080x1920_JP.mp4
  └── 260517_..._1200x628_JP.mp4
260520_..._1080x1080_JP/
  ├── 260520_..._1080x1080_JP.jpg
  └── 260520_..._1080x1920_JP.jpg
```

## 🎨 결과 색상

| 색상 | 상태 | 의미 |
|------|------|------|
| 🟢 초록 | 예정 / 완료 | 변경 대상 |
| 🔵 파랑 | 기준 | 사이즈 매칭의 기준 파일 (변경 안 함) |
| 🟡 노랑 | 건너뜀 | 변경 불필요 (이미 처리됨 등) |
| 🔴 빨강 | 오류 | 변경 실패 (같은 이름 존재 등) |

---

## ⚠️ 안전장치

- **미리보기 필수**: 실제 변경 전 결과를 먼저 확인
- **실행 취소(↩)**: 마지막 변경을 즉시 원복
- **중복 방지**: 이미 변환된 파일은 자동 건너뜀
- **이름 충돌 방지**: 같은 이름이 이미 있으면 오류 처리 (덮어쓰기 없음)

---

## 💡 추천 활용 흐름

```
① 제각각 이름 파일 받으면
  → 사이즈 매칭 (자동) 으로 네이밍 통일
② 네이밍 통일 완료 후
  → 폴더 그룹핑으로 소재별 폴더로 묶기
③ 드라이브 적재
```
